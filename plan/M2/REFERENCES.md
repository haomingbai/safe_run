# M2 技术选型与参考资料

## 1. 技术选型

- 路径处理：Rust `std::fs::canonicalize` + 安全前缀匹配。
- 挂载执行：优先结构化调用（避免自由拼接命令）。
- 测试：恶意样例驱动 + 回归快照。

## 2. 选型理由

- 路径规范化是阻断目录穿越和软链绕过的核心。
- 可回滚挂载计划能降低半成功状态带来的安全风险。
- 样例驱动测试可快速沉淀安全规则。
